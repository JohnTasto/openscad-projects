# Summary

After looking to make a mount for a BLTouch, I found a couple existing models of the BLTouch here on thingiverse but they were either too simple for my purposes or had dimensions that looked incorrect.
This model is measured from a physical BLTouch using calipers.
The Fusion 360 version also has a movable pin to help in designing mounts.
