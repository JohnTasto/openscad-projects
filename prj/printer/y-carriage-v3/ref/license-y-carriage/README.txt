# Summary

Y-axis carriage replacement for replicator 1, 2, or clones. There are two versions one for lm8suu (original) and for the lm8uu, great if upgrading to the igus type linear bearings like "IGUS RJ4JP-01-08".

# Print Settings

Printer Brand: MakerBot
Printer: MakerBot Replicator 2
Rafts: Doesn't Matter
Supports: Yes
Resolution: Normal
Infill: 20

# Post-Printing

## Primary front and back part

The primary carriage is printed in two parts. Use 4x M3 6mm or 8mm screws and super glue to attach the pieces to one and other.
